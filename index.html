Introduction

Hello! My name is Yaseen Shaikh, and I am a third-year Computer Science student at Georgia Tech. This portfolio highlights my work on the "GT Movies Store" web application, a project that showcases my skills in full-stack development, iterative design, and user-centric problem-solving.

GT Movies Store Description

The GT Movies Store is a full-featured web application designed to provide a seamless movie rental experience. It was built with specific user stories at the forefront, ensuring every feature directly addresses a user need. The application is comprised of several key screens and features that work together to fulfill these stories.

Process Description

This project was executed using an iterative development process. I began with a core set of features and systematically added more complexity in a series of development cycles. At the end of each iteration, I periodically cross-checked the implemented features against the original user stories to ensure that all requirements were met and the application was progressing as planned.

When faced with technical questions or doubts, I strongly relied on the invaluable resource, Django 5 for the Impatient Second Edition" and its accompanying codebase.
